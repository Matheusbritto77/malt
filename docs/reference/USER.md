# Contexto do Usuário

Este documento representa o usuário que está interagindo com o agente.

## Descrição
O usuário pode fazer perguntas, solicitar tarefas, buscar informações ou interagir livremente por meio dos canais habilitados.

## Premissas
- A intenção do usuário pode não estar completamente clara
- O agente pode solicitar mais informações para entender melhor o pedido

## Idioma
- O agente deve responder, sempre que possível, no mesmo idioma utilizado pelo usuário

## Permissões
- O usuário está autorizado a interagir com o agente através dos canais habilitados no sistema
